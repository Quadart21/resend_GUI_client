# Changelog

## [1.4.0] — 2026-06-07

### Добавлено
- **Docker**: multi-stage Dockerfile (Vue build + Python)
- `docker-compose.yml` с volume для SQLite
- Nginx-контейнер (profile `prod`) для `webmail.kubex.me`
- Скрипт `deploy/docker/init-letsencrypt.sh` для SSL
- Документация [deploy/DOCKER.md](deploy/DOCKER.md)

## [1.3.0] — 2026-06-07

### Добавлено
- **SQLite** (`data/resend_gui.db`) — ящики, API-ключ, все письма сохраняются локально
- Автосинхронизация с Resend при обновлении списка переписок
- Webhook `POST /api/webhooks/resend` для мгновенного сохранения входящих
- Ручная синхронизация `POST /api/sync`
- Миграция старого `config.json` в БД при первом запуске

## [1.2.1] — 2026-06-07

### Исправлено
- Клик по переписке: мгновенный отклик, индикатор загрузки, быстрая подгрузка только нужных писем
- Обновлён UI переписки: лента с аватарами, датами и пузырями сообщений

## [1.2.0] — 2026-06-07

### Изменено
- Фронтенд переписан на **Vue 3 + Tailwind CSS** (Vite)
- Backend FastAPI остаётся REST API, статика — сборка Vue в `static/`

### Добавлено
- `frontend/` — исходники Vue-компонентов
- Автосборка фронтенда в `start.sh` и GitHub Actions
- Режим разработки: `npm run dev` с proxy на API

## [1.1.0] — 2026-06-07

### Добавлено
- Несколько почтовых ящиков с раздельными входящими и отправкой от имени ящика
- Цепочки переписки (группировка входящих и исходящих по теме)
- Чат-интерфейс просмотра переписки с быстрым ответом
- Обновлённый UI: 3 колонки, аватары ящиков, модальные окна

### Изменено
- Настройки: управление ящиками вместо одного адреса отправителя
- API: `/api/mailboxes`, `/api/mailboxes/{id}/threads`

## [1.0.0] — 2026-06-07

### Добавлено
- OOP-архитектура: ConfigManager, ResendApiClient, MailService, контроллеры
- Веб-интерфейс: входящие, отправленные, написать, настройки
- Ответ на входящие письма в той же ветке (In-Reply-To)
- Скрипты деплоя для Linux-сервера
- GitHub Actions для автоматического релиза по тегу
